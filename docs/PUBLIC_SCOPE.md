# Public Scope

This public showcase repository exists to present Benthium professionally under the Hostinium company identity while protecting the private working product.

## Included Public Material

- product positioning
- business scope and target use cases
- high-level technology summary
- architecture direction
- safe product and portfolio context

## Excluded Public Material

- private implementation details
- environment configuration
- internal secrets and credentials
- production datasets
- customer-specific workflows
- delivery documents and operational playbooks
- logic detail that would make the product easier to reproduce

## Publishing Standard

Any content added to this repository should answer one of these questions safely:

- what problem does the product solve
- what kind of organization is it built for
- what is the system shape at a high level
- why does the product matter commercially or operationally

If content answers how the private system is implemented in a way that reduces defensibility, it does not belong here.
